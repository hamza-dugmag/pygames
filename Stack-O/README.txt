### THE FILE TO BE RUN ###
main.py --> main file; mangages menu system and when to quit
### THE FILE TO BE RUN ###

init.py --> init file; defines all shared variables and functions
button --> button class; generates button with text aligned within based on arguments
menu.py --> main menu file; manages main menu states and initiates log in / register
users.py --> user class; handles database handling, changes stats and users
dashboard.py --> dashboard class; manages dashboard menu states and initiates game
game.py --> game class; main gameplay

players.dat --> players database; contains users created with respective password and point stats
